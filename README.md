# React + Spring Boot + MySQL Employee Management System

A full-stack Employee Management System built using **React.js (Frontend)** and **Spring Boot (Backend)** with **MySQL** database integration.

This application allows users to manage employees with full CRUD functionality and pagination support.

## 🚀 Features

- ✅ Create Employee
- ✅ View All Employees
- ✅ Update Employee
- ✅ Delete Employee
- ✅ Pagination (5 records per page)
- ✅ RESTful API architecture
- ✅ MySQL database integration
- ✅ Clean UI using React Bootstrap
-
